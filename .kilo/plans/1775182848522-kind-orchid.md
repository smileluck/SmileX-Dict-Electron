# 用户个人设置功能开发计划

## 项目概述

为SmileX Dict应用开发用户个人设置功能模块，包括设置页面、配置管理、签到逻辑优化和学习流程改进。

## 功能需求

### 1. 个人设置页面
- **入口位置**: 登录后顶部导航栏的用户名下拉菜单，新增"个人设置"选项
- **配置项**:
  - 单词背诵模式：英英模式 / 中英模式（单选按钮组）
  - 用户信息修改：用户名
  - 每日学习新词量：不包含复习单词的每日新词学习目标
- **交互**: 保存按钮，保存后更新配置并显示成功提示

### 2. 签到功能优化
- **新规则**: 需要完成每日所有单词学习（复习+新词），才能签到
- **判断逻辑**: 当日所有待学习单词（nextReviewDate ≤ 今天）都已完成

### 3. 单词学习顺序优化
- **优先级**: 复习单词 → 新词
- **筛选规则**:
  1. 先筛选需要复习的单词（status !== 'mastered' && nextReviewDate ≤ 今天）
  2. 按复习优先级排序（错误的单词优先）
  3. 然后再学习新词（status === 'new'）

### 学习逻辑优化

#### 签到条件判断

```typescript
// src/features/panel/panelSlice.ts

// 新增action
canSignInToday: (words: WordItem[]) => {
  const today = new Date().toISOString().split('T')[0]
  const pendingWords = words.filter(w =>
    w.status !== 'mastered' &&
    w.nextReviewDate.split('T')[0] <= today
  )
  return pendingWords.length === 0
}
```

#### 单词队列排序

```typescript
// src/routes/PracticeWords.tsx

// 修改筛选逻辑
const queue = useMemo(() => {
  const today = new Date().toISOString().split('T')[0]
  let filtered = words

  if (selectedDictId) {
    // ... 字典筛选逻辑
  }

  // 基础筛选：需要学习的单词
  filtered = filtered.filter(w =>
    w.status !== 'mastered' &&
    w.nextReviewDate.split('T')[0] <= today
  )

  // 排序优先级：
  // 1. 错误单词（status === 'wrong'）
  // 2. 复习单词（status === 'new' && nextReviewDate < 今天）
  // 3. 新词（status === 'new' && nextReviewDate === 今天）
  return filtered.sort((a, b) => {
    if (a.status === 'wrong' && b.status !== 'wrong') return -1
    if (a.status !== 'wrong' && b.status === 'wrong') return 1

    const aIsReview = a.nextReviewDate.split('T')[0] < today
    const bIsReview = b.nextReviewDate.split('T')[0] < today
    if (aIsReview && !bIsReview) return -1
    if (!aIsReview && bIsReview) return 1

    return 0
  })
}, [words, selectedDictId])
```

## 数据库迁移SQL

```sql
-- 创建用户配置表
CREATE TABLE IF NOT EXISTS user_settings (
    userId TEXT PRIMARY KEY,
    username TEXT UNIQUE NOT NULL,
    practiceMode TEXT DEFAULT 'zh-en',
    dailyNewWordTarget INTEGER DEFAULT 20,
    lastUpdateAt DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (userId) REFERENCES users(id) ON DELETE CASCADE
);

-- 为现有用户创建默认配置
INSERT INTO user_settings (userId, username, practiceMode, dailyNewWordTarget)
SELECT id, username, 'zh-en', 20
FROM users
WHERE id NOT IN (SELECT userId FROM user_settings);
```
