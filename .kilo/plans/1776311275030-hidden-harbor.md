# SmileX Dict UI Optimization & Beautification Plan

## Design Direction: Modern Glassmorphism
- Frosted glass effects (backdrop-blur, semi-transparent backgrounds)
- Subtle gradient accents (brand color gradients, mesh backgrounds)
- Soft layered shadows
- Smooth micro-interactions and transitions
- Clean typography hierarchy
- Premium dark mode with true native Tailwind `dark:` variants

---

## Phase 1: Design Foundation

### 1.1 Tailwind Config Updates (`tailwind.config.js`)
- Add glassmorphism utility classes: `backdrop-blur-*`, custom `bg-glass-*` colors
- Extend theme with CSS variable-based color system for seamless light/dark switching
- Add new animation keyframes: `float`, `shimmer`, `slide-up`, `scale-in`
- Add box-shadow tokens: `shadow-glass`, `shadow-card`, `shadow-elevated`
- Add gradient presets: `bg-gradient-brand`, `bg-gradient-warm`, `bg-gradient-cool`

### 1.2 CSS Variable System (`src/index.css`)
- Define CSS custom properties for all colors under `:root` and `.dark`
- Remove ALL existing `.dark .bg-white`, `.dark .text-gray-*` global overrides (lines 81-153)
- Replace with proper `dark:` Tailwind variants in each component
- Add glassmorphism base styles: `.glass` (bg-white/70 backdrop-blur-xl border border-white/20)
- Add gradient mesh background patterns
- Add subtle noise texture overlay for glass backgrounds
- Update scrollbar styling for dark mode

### 1.3 Dark Mode Migration
Every component file needs to migrate from relying on global `.dark` CSS overrides to explicit `dark:` Tailwind variants. The pattern for each element:
- `bg-white` → `bg-white dark:bg-gray-800`
- `bg-gray-50` → `bg-gray-50 dark:bg-gray-900`
- `text-gray-900` → `text-gray-900 dark:text-gray-100`
- `text-gray-600` → `text-gray-600 dark:text-gray-400`
- `border` / `border-gray-200` → `border-gray-200 dark:border-gray-700`
- `bg-brand-100` → `bg-brand-100 dark:bg-brand-900/30`
