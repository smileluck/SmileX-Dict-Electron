# SM2 优化与英语专业学习功能增强计划

## 项目概述
优化现有的SM2单词记忆算法，并为英语专业学生提供更有效的学习功能和深度词汇学习体验。

## 现状分析

### 当前优势
- ✅ 完整的SM2算法实现（记忆因子1.3-2.5，间隔递增，质量评分0-5）
- ✅ 多种学习模式（打字拼写、快速确认、中英、英英）
- ✅ 完善的用户进度跟踪和统计系统
- ✅ 跨平台支持（Web、移动、桌面）
- ✅ 用户设置系统

### 现有SM2算法实现位置
- **前端逻辑**: `src/features/words/wordsSlice.ts:71-112`
- **数据模型**: `server/models.py:24-35`
- **练习界面**: `src/routes/PracticeWords.tsx:134-231`

## 优化目标

### 1. SM2算法优化
- **精细化质量评分**: 从现有的0-5分扩展到0-6分，增加"完全掌握"选项
- **记忆因子动态调整**: 根据单词难度和重要性调整E-Factor范围
- **学习疲劳度管理**: 监测学习效率，动态调整复习密度
- **智能间隔调度**: 基于个人学习模式和表现优化间隔时间

### 2. 英语专业学习功能增强
- **词根词缀学习系统**: 建立词根词缀数据库，支持单词构成分析
- **学术词汇分类**: 按学科、词频、难度分类的专业词汇
- **深度词汇学习**: 包含同义词辨析、搭配用法、语域分析
- **写作词汇积累**: 分类收集高级替换词和表达方式
- **语法结合练习**: 将单词学习与语法结构结合

### 3. 高级学习功能
- **个性化学习路径**: 基于目标和进度推荐学习内容
- **词汇网络分析**: 展示单词之间的语义关联
- **智能错词分析**: 识别错误模式和薄弱环节
- **发音练习系统**: 集成语音识别和发音纠正

#### 1.3 优先级队列优化
```typescript
// 智能排序算法
function sortWordsByPriority(words: EnhancedWordItem[]): EnhancedWordItem[] {
  return words.sort((a, b) => {
    const aPriority = calculatePriority(a);
    const bPriority = calculatePriority(b);
    return aPriority - bPriority;
  });
}

function calculatePriority(word: EnhancedWordItem): number {
  let priority = 0;
  
  // 1. 错词优先级最高
  if (word.status === 'wrong') return 1000;
  
  // 2. 基于重要性和难度的权重
  priority += word.importance * 100;
  priority += word.difficulty * 50;
  
  // 3. 基于疲劳度的惩罚
  priority -= word.fatigueFactor * 20;
  
  // 4. 基于复习间隔的紧急程度
  const daysUntilReview = calculateDaysUntilReview(word);
  priority += Math.max(0, 7 - daysUntilReview) * 10;
  
  return priority;
}
```
