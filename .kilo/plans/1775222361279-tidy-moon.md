# 英英/中英模式需求修正计划

## 需求理解

**当前错误实现：**
- `zh-en`（中→英）：显示中文释义 → 打字输入英文单词
- `en-en`（英→英）：显示英文单词 + 例句 → 四选一选择题（中文选项）

**正确需求：**
- 中英模式和英英模式是**互斥的释义语言设置**，控制学习过程中出现的讲解语言
- **中英模式**：显示中文释义"糖" → 打字输入英文单词 "sugar"
- **英英模式**：显示英文释义 "It's what makes candy taste sweet." → 打字输入英文单词 "sugar"
- 两种模式答题方式统一为**打字输入英文单词**

## 需要修改的内容

#### 答题方式（en-en 模式）

#### 合并 zh-en 和 en-en 的答题逻辑

由于两种模式答题方式相同（打字输入英文），可以合并为一个 `submitRecall` 函数：

```typescript
const submitRecall = () => {
  if (!current) return
  const wasNew = current.status === 'new'
  const userInput = input.trim().toLowerCase()
  const correctTerm = current.term.toLowerCase()
  const ok = userInput === correctTerm

  let quality: number
  const responseTime = responseTimeRef.current > 0 ? Math.round(performance.now() - responseTimeRef.current) : 0

  if (ok) {
    quality = responseTime < 3000 ? 5 : 4
  } else {
    const similarity = calculateSimilarity(userInput, correctTerm)
    quality = similarity > 0.8 ? 2 : similarity > 0.5 ? 1 : 0
  }

  dispatch(reviewWord({
    wordId: current.id,
    quality,
    responseTime,
    learningContext: 'recall'
  }))
  // ...event tracking...

  setResult(ok ? 'correct' : 'wrong')
}
```
