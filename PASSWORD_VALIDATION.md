# 密码校验显示功能

## 功能概述

为登录/注册表单添加了实时的密码强度指示器和要求检查显示，帮助用户创建符合安全要求的密码。

## 功能特性

### 1. **密码强度指示器**

**显示位置**：注册时密码输入框下方

**视觉反馈**：
- 进度条显示密码强度百分比
- 颜色编码：
  - 🔴 红色 - 弱（< 40%）
  - 🟡 黄色 - 中（40-60%）
  - 🔵 蓝色 - 良（60-80%）
  - 🟢 绿色 - 强（≥ 80%）

**强度计算规则**：
```typescript
总分 100 分：
- 长度 ≥ 8 字符：+20 分
- 长度 ≥ 12 字符：额外 +20 分
- 包含大写字母：+20 分
- 包含小写字母：+20 分
- 包含数字：+20 分
- 包含特殊字符：+20 分
```

### 2. **密码要求检查**

**显示位置**：注册时密码强度条下方

**要求列表**（5项）：
1. ✓ 8-64 个字符
2. ✓ 大写字母（A-Z）
3. ✓ 小写字母（a-z）
4. ✓ 数字（0-9）
5. ✓ 特殊字符（!@#$%^&*()_+-=[]{}|;:,.<>/?）

**实时状态更新**：
- ✓ 已满足：绿色背景 + 勾号
- ○ 未满足：灰色背景 + 圆圈

**布局**：
- 桌面端：3列网格布局
- 移动端：2列网格布局

### 3. **登录页面提示**

登录时在密码框下方显示简短提示：
```
密码要求：8-64个字符，包含大小写字母、数字和特殊字符
```

## 用户体验

### 注册流程

1. 用户输入密码
2. 实时显示：
   - 密码强度条动画更新
   - 要求列表实时标记满足项
3. 点击提交时验证所有要求
4. 显示具体的错误提示

**场景1：输入弱密码**
